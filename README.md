# ApyHub Explorer

Browse, search, and integrate ApyHub APIs directly from Voiden. Discover 200+ production-ready APIs across AI, data processing, file conversion, and more — with live documentation and one-click request scaffolding.

## Features

- **API Catalog** — Browse featured and categorized APIs from the ApyHub marketplace
- **Search** — Find APIs by keyword with debounced live results
- **Category Icons** — Color-coded category badges and icons matching the ApyHub website
- **API Documentation** — View endpoint routes, methods, request examples, and sample responses
- **One-Click Testing** — Copy cURL commands or scaffold requests directly into Voiden
- **Paginated Browsing** — Navigate through the full catalog with pagination controls

## Categories

Artificial Intelligence, Data Extraction, Data Validation, E-commerce Quick Tools, File Conversion, File Manipulation, File Security, HR, Image Processing, Marketing, SEO, Smart Generation, Standard Data, Travel, and more.

## Usage

1. Install the extension from the Voiden extension browser
2. Click the **ApyHub** button in the status bar to open the API catalog
3. Browse featured APIs or search by keyword
4. Click any card to view full documentation
5. Use **Copy curl** or **Test** to scaffold requests into your workspace

## Development

```bash
npm install
npm run build        # Build dist/main.js + dist/manifest.json
npm run dev          # Watch mode
```
